# 話題のAIコード生成ツール、結局どれが一番使えるの？ガジェットレビュアーが忖度なしで検証してみた

結論から言うと、プログラミング未経験の僕が「AIコード生成」を試した結果、**これ一択！と断言できるほど単純じゃなかった**。

いやマジで。巷で「AIがコード書いてくれるらしいよ！」って噂は聞いたことある人も多いと思う。僕もご多分に漏れずその一人。プログラミングの「プ」の字も知らない人間が、いきなりAIコード生成ツールに手を出してみたんだ。

今回比較したのは、現時点でAI界の三銃士と言われる「ChatGPT」「Claude」「Gemini」の3つ。まるで最新スマホをレビューするみたいに、忖度なしで徹底的に使い倒してみた。それぞれのAIがどんなコードを吐き出して、どれが一番プログラミング初心者の僕にとって『バイブコーディング』（書いてて気持ちいい、楽しい状態ね！）しやすかったのか、マジメに検証していくよ。

この記事を読めば、「AIにコード作ってもらう」って聞いたけど、結局どれを使えばいいか分からないプログラミング初心者、そして何ならプログラミング自体に触れたことすらない人でも、自分に合ったAIを見つけられるはず。

さて、早速検証結果を見ていこうか。

## 【検証1】「初めてのWebサイト」を作らせてみた！初心者が『バイブコーディング』しやすいのはどれだ？

まずは、プログラミングの登竜門とも言える「自己紹介サイト」作成に挑戦してもらった。要求はシンプル。

```text
プログラミング未経験者です。
HTMLとCSSを使って、シンプルな自己紹介サイトを作成したいです。
自己紹介の内容は「私の名前はAI太郎です。趣味はAIとの会話です。夢はAIと共存する社会です。」とします。
デザインは、背景を薄いグレーにして、文字は黒、中央寄せでお願いします。
コードの解説も丁寧にお願いします。
```

このプロンプト（AIへの指示文のことね）をそれぞれのAIに投げかけてみた。評価のポイントは3つ。

1.  **コードの分かりやすさ**: 初心者でも「なるほど！」と理解できるコードか。
2.  **エラーの少なさ**: コピペしてすぐに動くか。修正の手間はどれくらいか。
3.  **『バイブコーディング』しやすさ**: これが一番重要。書いてて楽しいか、挫折しそうにならないか。

さあ、各AIの性能を見ていこう。

## ChatGPT: 安定感の優等生？でも、たまに「お前、本当にわかってる？」となる瞬間

まずはみんな大好き、ChatGPT。言わずと知れたAI界の王様だ。期待値も高まる。

プロンプトを投げると、サッとHTMLとCSSを生成してくれた。そして、それぞれのコードについて「これはHTMLです」「これはCSSです」という丁寧な説明が続く。

:::details ChatGPTが生成したHTMLコード
```html:index.html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>AI太郎の自己紹介</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="container">
        <h1>AI太郎の自己紹介</h1>
        <p>私の名前はAI太郎です。</p>
        <p>趣味はAIとの会話です。</p>
        <p>夢はAIと共存する社会です。</p>
    </div>
</body>
</html>
```
:::

:::details ChatGPTが生成したCSSコード
```css:style.css
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0; /* 薄いグレー */
    color: #333; /* 文字色を黒に近い色に */
    display: flex;
    justify-content: center; /* 水平方向の中央寄せ */
    align-items: center; /* 垂直方向の中央寄せ */
    min-height: 100vh; /* 画面全体の高さを利用 */
    margin: 0;
}

.container {
    text-align: center; /* テキストの中央寄せ */
    padding: 20px;
    background-color: #fff; /* コンテナの背景を白に */
    border-radius: 8px; /* 角を丸くする */
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1); /* 影をつける */
}

h1 {
    color: #007bff; /* タイトル色をアクセントに */
}
```
:::

コピペしてブラウザで開いてみると……おお、ちゃんと動いた！ 薄いグレーの背景に黒い文字、中央寄せ。要求通りのシンプルな自己紹介サイトができあがった。

説明もそこそこ丁寧で、各タグ（`<h1>`とか`<p>`とかね）やCSSのプロパティ（`background-color`とか`text-align`とか）が何をしているのかを教えてくれる。

ただ、個人的にちょっと気になったのは、たまに「え、そこまで説明いる？」ってくらい回りくどい言い回しがあったり、「これ、本当に初心者が一番最初に覚えるべきことか？」みたいな、ちょっとだけ複雑なCSS（`display: flex`とか`min-height: 100vh`とかね）をサラッと出してくること。

もちろん、それらの技術はWebサイトを作る上で非常に便利なんだけど、右も左も分からない初心者にとっては、「うわ、なんか難しそう…」って一瞬戸惑うポイントになるかもしれない。

:::message
**補足情報：`min-height: 100vh`って何？**

これは「画面の高さの100%分、最低限の高さを持たせる」って意味。Webサイトをスクロールしなくても、画面いっぱいにコンテンツを表示したい時とかによく使われる。初心者にはちょっと高度なCSSかもね。
:::

総じて優等生なんだけど、たまに「もうちょい初心者の気持ちになってくれよ！」って思う瞬間があったのが正直な感想だ。

## Claude: 丁寧な説明が光る新人！でも、ちょっと『おしゃべり』が過ぎるかも？

次に試したのは、最近話題のClaude。特に長文読解や丁寧な回答に定評があるらしい。これは初心者にとって嬉しいんじゃないか？と期待してプロンプトを投げた。

すると、ChatGPTと同じくHTMLとCSSを生成。そして、その説明文がとにかく長い！

:::details Claudeが生成したHTMLコード（一部）
```html:index.html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>AI太郎の自己紹介ページ</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="container">
        <h1>AI太郎の自己紹介</h1>
        <p>私の名前はAI太郎です。</p>
        <p>趣味はAIとの会話です。</p>
        <p>夢はAIと共存する社会です。</p>
    </div>
</body>
</html>
```
:::

:::details Claudeが生成したCSSコード（一部）
```css:style.css
body {
    font-family: 'Arial', sans-serif; /* 一般的なフォントを指定 */
    background-color: #f5f5f5; /* 薄いグレー */
    color: #333; /* 濃いめのグレーで読みやすく */
    display: flex;
    justify-content: center; /* 水平方向の中央揃え */
    align-items: center; /* 垂直方向の中央揃え */
    min-height: 100vh; /* ビューポート（画面）いっぱいの高さ */
    margin: 0;
    padding: 20px; /* 全体の余白 */
    box-sizing: border-box; /* パディングを含めてサイズ計算 */
}

.container {
    background-color: #ffffff; /* コンテナの背景は白 */
    padding: 30px; /* コンテナ内の余白 */
    border-radius: 10px; /* 角を少し丸く */
    box-shadow: 0 5px 15px rgba(0, 0, 0, 0.1); /* 軽い影で立体感を */
    text-align: center; /* テキストを中央揃え */
    max-width: 600px; /* コンテナの最大幅 */
    width: 90%; /* レスポンシブ対応 */
}
```
:::

コード自体はChatGPTとほぼ同じレベルで、ちゃんと動いた。ここまでは問題なし。
ただね、説明が丁寧すぎて、コードを生成するまでに前置きが長いし、コードの後も延々と説明が続くんだ。

例えば、HTMLの`<head>`タグの説明で「ここに書かれた情報はブラウザには直接表示されませんが、ウェブサイトの動作や表示に重要な役割を果たします」とか、CSSの`font-family`で「もし最初のフォントが利用できない場合、次のフォントが試されます」とか、とにかく懇切丁寧。

親切心はわかるんだけど、正直「早くコード見せてくれ！」って思っちゃう。おしゃべりな友達と一緒で、話が面白いんだけど、本題になかなかたどり着かない感じ。

プログラミング初心者が「とりあえず動くコードが欲しい！」って時には、この『おしゃべり』がちょっと邪魔に感じるかもしれない。もちろん、じっくり学びたい人には最高だと思うけどね。

:::message alert
**注意点：AIが生成したコードは鵜呑みにしない！**

どんなに優秀なAIでも、完璧なコードを吐き出すわけじゃない。特に初心者のうちは、AIが生成したコードをそのままコピペするだけでなく、少しでも「これ、何してるんだろう？」と思ったら、**必ず自分で調べてみる**のが上達への近道だ。
:::
